static <TextendsJavaPlugin> T initPlugin ( String  name,
String  version,
Class< T >  pluginClass 
)
static

Instantiate and init a non-mock Bukkit plugin.

The plugin is initialized without commands. The returned instance is not yet enabled.

The plugin requires a constructor with private or greater access with the following arguments:

  • org.bukkit.plugin.java.JavaPluginLoader
  • org.bukkit.plugin.PluginDescriptionFile
  • java.io.File
  • java.io.File

The arguments passed into the constructor should be forwarded to the protected superclass org.bukkit.plugin.java.JavaPlugin constructor of the same signature.

Parameters
nameThe name to initialize the plugin with.
versionThe version to initialize the plugin with.
pluginClassThe plugin class.
<T>The plugin type.
Returns
The plugin instance.
